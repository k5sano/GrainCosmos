## Context to Preserve

**Key Decisions:**
- Plugin type: Audio Effect (Tape Saturator)
- Core concept: Warm tape saturation with controllable degradation (wow/flutter/dropout/noise)
- AGE parameter must remain musical even at maximum—never "stupid" sounding
- Visual aesthetic: Premium vintage tape machine, skeuomorphic brass controls, earth tones
- 3-parameter design: DRIVE (saturation), AGE (degradation), MIX (dry/wet)

**Design Details (v3 - FINAL):**
- Window: 500x450px (vertically centered content block)
- Background: Radial gradient (#4a3a2a to #2a1a0a), horizontal scan lines, heavy vignette
- VU meter: Deep recessed panel (#1a0a00), bronze frame (#3a2a1a), warm gold needle with glow
- Title: "TAPE AGE" in Helvetica Neue (light 300 weight, 0.4em letter spacing)
- Knobs: 90px dark bronze with brushed metal centers, gold indicator lines, FIXED lighting (realistic)
- Labels: Directly below knobs (DRIVE, AGE, MIX), bronze color (#c49564), medium weight (500)
- Typography: Helvetica Neue (classic, geometric, clean sans-serif)
- Lighting: Static shadows/highlights on knob body, only indicator rotates (physically realistic)

**Technical Notes:**
- WebView-based UI (JUCE WebBrowserComponent)
- JavaScript ↔ C++ communication via `window.juce.postMessage()`
- Parameter bindings handle automation sync
- VU meter updates at 60 Hz via timer callback
- All parameters normalized 0.0-1.0 (JUCE convention)
